bool LimeBrokerage::StrategyStudio::MarketModels::AccountFees::passes_through_exchange_fees

Indicates whether the account's broker passes through exchange fees

If this flag is set to true, Strategy Studio will try to listen to whether a fill has added or removed liquidity, and will apply the exchange's posting/taking fees/rebates as appropriate
