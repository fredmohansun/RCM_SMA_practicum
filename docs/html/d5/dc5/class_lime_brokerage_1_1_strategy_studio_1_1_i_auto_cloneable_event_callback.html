Detailed Description

template<typename Derived>
class LimeBrokerage::StrategyStudio::IAutoCloneableEventCallback< Derived >

Uses the CRTP pattern to provide auto implementation of IClonebleEventCallback's clone() method via the derived class's copy constructor.

To use this class, simply define a class like "class MyEventCallback : public IAutoCloneableEventCallback<MyEventCallback>{}"
