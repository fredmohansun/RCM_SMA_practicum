bool LimeBrokerage::StrategyStudio::QuoteEventMsg::FeedOnlyHasNBBOQuotes ( ) const
inline

Returns true if data source is only sending quote ticks that affect the NBBO

This is true of some Consolidated (L1) datasets. In this mode Strategy Studio copies the NBBO to the top_quote() rather than maintaining a top_quote() that represents the sum of liquidity from all market centers at top prices. This is because with only the NBBO datapoints we can't accurately maintain the state of each market center's quotes.
